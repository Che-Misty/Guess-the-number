# Игра "Угадай число"
Консольная игра на отгадывание числа.

## Возможности
- 3 уровня сложности: Easy, Medium, Hard
- Подсчет попыток и ведение списка введенных чисел за игру
- Отображение подсказок:
| Горячо  | Загаданное число в пределах 5 единиц          |
| ------- | --------------------------------------------- |
| Тепло   | Загаданное число в пределах от 6 до 15 единиц |
| Холодно | До загаданного числа больше 15 единиц.        |
- История игр сохраняется в JSON-файл

## Запуск

### Требования

- Go 1.25+

### Установка

```
git clone https://github.com/CheMisty-98/guess-the-number
cd guess-the-number
go run .
```

## Как играть

1. Выбери уровень сложности
2. Угадывай загаданное число
3. Получай подсказки: больше / меньше
4. Победи за минимальное количество попыток

## Уровни сложности

| Уровень | Диапазон | Макс. попыток |
|---------|----------|---------------|
| Лёгкий  | 1–50     | 15            |
| Средний | 1–100    | 10            |
| Сложный | 1–200    | 5             |
